# Dogonauts Monolith v2.0

Monolito Node.js/TypeScript con SQLite + node-schedule.

## Scripts

- `npm run dev` → arranca en modo ts-node
- `npm run build` → compila a dist
- `npm start` → ejecuta build
- `npm run migrate`→ asegura el schema y (luego) importa CSV

## Docker

- `docker compose up --build -d`
